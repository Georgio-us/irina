# Дизайн-система и Руководство по верстке раздела «Библиотека знаний»

Этот документ содержит описание архитектуры верстки, дизайн-токенов (цветов, шрифтов, скруглений, отступов) и сетки, используемых в обновленном блоке табуляций раздела «Библиотека знаний» (`.courses`) на сайте. 

Документ создан для разработчиков и дизайнеров с целью сохранения преемственности стилей при дальнейшей доработке интерфейсов.

---

## 1. Сетка и Структура Лейаута (Grid & Layout)

В основе верстки лежит жесткая **12-колоночная сетка** с локальными адаптациями под каждый тип контента.

### Основные параметры сетки:
- **Ширина контейнера**: Макет зафиксирован в обертке `.page-wrapper` на ширине **`1440px`** с центрированием `margin: 0 auto`.
- **CSS Grid**: Элементы табуляций используют `display: grid; grid-template-columns: repeat(12, 1fr);`.
- **Межколоночный интервал (Gap)**:
  - Для блоков «Вебинары» и «Научная база»: `gap: 24px 20px` (строки / колонки).
  - Для блока «Книга»: `column-gap: 24px`.
- **Внутреннее выравнивание**: Большинство колонок деталей растягиваются по высоте контента обложек (`align-items: stretch`).

### Разметка по колонкам в табах:
1. **Авторские программы (`#grid-programs`)**:
   - Состоит из двух карточек, каждая занимает **6 колонок** (`grid-column: span 6`).
2. **Вебинары (`#grid-webinars`)**:
   - Главная обложка: **6 колонок** (`span 6`).
   - Детализация вебинара: **4 колонки** (`span 4` с 7 по 10 колонки). Колонки 11 и 12 остаются свободными для асимметрии.
   - Нижний ряд карточек: 4 карточки по **3 колонки** каждая (`span 3`).
   - Нижняя цитата: на всю ширину **12 колонок** (`span 12`).
3. **Книга (`#grid-books`)**:
   - Обложка книги: **6 колонок** (`span 6`).
   - Описание книги: **4 колонки** (`span 4` с 7 по 10 колонки).
   - Блок превью-карточек справа: **2 колонки** (`span 2` с 11 по 12 колонки).
4. **Научная база (`#grid-science`)**:
   - Обложка: **6 колонок** (`span 6`).
   - Детализация базы: **4 колонки** (`span 4` с 7 по 10 колонки).
   - Средний ряд (Cambridge CTA Banner): на всю ширину **12 колонок** (`span 12`).
   - Нижний ряд карточек: 3 карточки по **4 колонки** каждая (`span 4`).

## 2. Цветовая Палитра (Color Palette)

Для разделов «Шапка» (`.header`), «Главный экран» (`.hero`), «Библиотека знаний» (`.courses`) и «Результаты клиентов» (`.testimonials`) реализована локальная фиолетовая цветовая схема, переопределяющая базовые переменные темы:

```css
.header, .hero, .courses, .testimonials {
    --color-rose: #4C224C;        /* Глубокий фиолетовый (основной акцент) */
    --color-rose-dark: #381838;   /* Темно-фиолетовый (для ховеров) */
    --color-rose-light: #F5EBEF;  /* Нежно-лиловый/розовый (фоны блоков и иконок) */
}
```

### Дополнительные нейтральные цвета:
- **`#FAF8F6`** (Светло-кремовый): используется для фонов CTA-баннеров (например, Cambridge CTA).
- **`#FDFBF9`** (`--color-bg`): фоновый цвет секции.
- **`#EFE8E2`** (`--color-border`): цвет тонких разделительных линий.
- **`#333333`** (`--color-text-main`): основной цвет текста.
- **`#666666`** (`--color-text-muted`): приглушенный цвет описаний и метаданных.
- **`#FFFFFF`** (`--color-white`): белый цвет для текста на карточках и кнопок.

## 3. Типографика (Typography)

Используется сочетание антиквы для заголовков и гротеска для интерфейсного текста.

### Шрифтовые гарнитуры:
- **Serif (Заголовки)**: `'Playfair Display', Georgia, serif` (переменная `--font-serif`). Придает премиальный, классический вид.
- **Sans-serif (Интерфейс)**: `'Montserrat', system-ui, -apple-system, sans-serif` (переменная `--font-sans`). Читаемый и современный.

### Шрифтовая иерархия в Библиотеке:
- **Главные заголовки в табах (`.lib-title`)**: `26px` - `28px`, Serif, начертание `500` или `400`.
- **Подзаголовки/Метки категорий (`.lib-subtitle`)**: `10px` - `11px`, Sans-serif, начертание `700` или `800`, `text-transform: uppercase`, `letter-spacing: 0.15em`.
- **Названия карточек (`.science-card-title`, `.blog-card-title`)**: `14px` - `18px`, начертание `800` (Sans-serif) или `400` (Serif).
- **Основной текст (`.lib-desc`, `.science-card-desc`)**: `11px` - `13px`, Sans-serif, начертание `500` или `400`, `line-height: 1.4 - 1.55`.

## 4. Скругления углов (Border Radii)

Скругления обеспечивают мягкость интерфейса и упорядоченность блоков:
- **`20px`**: Крупные внешние плашки и горизонтальные баннеры с эффектом стекла (например, `.hero-stats-banner`, `.not-alone-card`).
- **`16px`**: Крупные обложки и контейнеры (например, `.science-cover-wrapper`, `.science-cta-wrapper`, `.video-testimonials-container`, внешняя колонка `.lib-left-col`).
- **`12px`**: 
  - Карточки нижних рядов (вебинары, научные статьи, видеоотзывы `.video-review-thumb`).
  - Слайдер-карточки отзывов (`.testimonial-slider-card`).
  - Фоновые плашки цитат (`.lib-quote-box`, `.webinar-bottom-quote`).
  - Все кнопки действий и контурные кнопки на сайте (`.btn-lib-action`, `.btn-lib-outline`, `.btn-science-cta`, `.btn-video-more`, `.btn-banner-action`, а также кнопки в `.header` и `.hero`).
- **`8px`**: Внутренние интерактивные элементы меньшего размера (поля ввода форм, превью-карточки, иконки соцсетей на главном экране `.hero-socials a`).
- **`50%` (Круг)**: Круглые бэджи иконок (`.feature-icon-circle`, `.stat-circle-icon`, `.banner-icon-circle`), кнопки управления плеером, круглые кнопки-стрелки (`.science-card-arrow`, `.slider-arrow`).

## 5. Отступы и Внутренние Расстояния (Paddings & Gaps)

Для сохранения чистоты макета все элементы внутри текстовых блоков выстроены через Flexbox с фиксированными промежутками:
- **Центральное управление отступами**: Все текстовые группы (Subtitle + Title + Desc + Quote/Buttons) обернуты в класс контента (например, `.science-details-content`) с правилом `gap: 12px`. Это исключает использование хаотичных `margin-bottom` у дочерних тегов.
- **Внутренние отступы баннеров**:
  - Баннеры CTA: `20px 32px` или `24px 32px` для сбалансированного распределения пространства.
  - Цитаты: `24px 32px`.
  - Карточки с фоновыми изображениями: `24px` со всех сторон.
- **Вертикальные линии-разделители**:
  - В блоке цитат Hero (`.hero-quote-card`): левая рамка `border-left: 1px solid rgba(255, 255, 255, 0.25)` с отступом `padding-left: 20px` формирует четкую вертикальную линию разграничения на фоне темного изображения.
  - В баннере преимуществ (`.banner-separator`): вертикальные разделители высотой `40px` с цветом `rgba(76, 34, 76, 0.1)`.

## 6. Анимации и Микро-взаимодействия (Animations & Transitions)

Для премиального ощущения от интерфейса добавлены мягкие ховер-эффекты:

### 1. Эффект приближения обложек (Zoom):
При наведении на карточки или блоки обложек фоновое изображение плавно масштабируется:
```css
.science-card-img, .science-cover-image {
    transition: transform 0.6s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
.science-card-item:hover .science-card-img {
    transform: scale(1.04);
}
```

### 2. Ховер кнопок-стрелок:
Белые круглые стрелки на карточках при наведении инвертируют цвета и слегка увеличиваются в размере:
```css
.science-card-arrow {
    transition: all 0.3s ease;
}
.science-card-item:hover .science-card-arrow {
    background-color: #4C224C;
    color: #FFFFFF;
    transform: scale(1.08);
}
```

### 3. Интерактивные кнопки плеера и вкладок:
- Вкладки верхнего меню плавно подчеркиваются линией при фокусе/клике.
- Кнопки плеера имеют ховер с легким изменением прозрачности оверлея или фонового тона.

## 7. Технические Примечания по Верстке
- **Адаптация обложек**: Все изображения обложек должны использовать свойство `object-fit: cover; width: 100%; height: 100%;` внутри фиксированных по высоте оберток. Это защищает дизайн от искажения пропорций картинок.
- **Отсутствие `!important`**: Верстка выполнена чисто, за счет вложенности селекторов и локального переопределения переменных, что гарантирует легкую поддержку.
- **Сброс кэша стилей**: При внесении изменений в файлы стилей не забывайте обновлять версию в подключении файла в `index.html` (например, `style.css?v=14`).
